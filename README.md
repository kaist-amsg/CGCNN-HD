# CGCNN-HD
The CGCNN-HD is a python code for dropout-based uncertainty quantification for stability prediction with CGCNN (by T. Xie et al.) developed by Jung group at KAIST.

Developers
----------

Dependencies
------------
-  Python3
-  Numpy
-  Pytorch == 0.4.1.post2 (CUDA 8.0)
-  Pymatgen
-  Sklearn
